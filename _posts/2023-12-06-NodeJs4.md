# Event Loop 심화 개념

* **Timer**
  - Event Loop의 시작을 알린다.
    + setTimeout()
    + setInterval()
   
* **Poll**
  - DB 관련 작업으로 인한 결과가 왔을 때 실행되는 콜백
  - HTTP 요청으로 인한 응답이 왔을 때 실행되는 콜백
  - 파일을 비동기로 읽은 후에 실행되는 콜백
   
* **Check**
  - Poll 단계 완료 후 콜백 실행 가능.
  - setImmediate() 사용하여 Queue에 지정된 것을 처리할 수 있음
