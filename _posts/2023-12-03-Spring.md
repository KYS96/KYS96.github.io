# MVC 패턴 요약

* (1) 입력
* (2) 처리
  - Controller
* (3) 출력
  - View
* (4) Model
  - 컨트롤러와 뷰 사이에서 데이터를 주고 받는 객체

# 출력 분리

* DispatcherServlet

  - 입력 요청이 들어오면 DispatcherServlet이 처리 후 해당 Controller에게 넘겨준다.
    * 이때 Model에 결과를 담아 Controller에 넘긴다.
  - Controller는 처리된 결과를 Model에 담는다.
  - DispatcherServlet은 작업 결과가 담긴 Model을 View로 넘겨준다.
